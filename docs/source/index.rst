Fastest-lap documentation
=========================

Fastest-lap is a vehicle dynamics simulator. It can be used to understand vehicle dynamics, to learn about driving techniques, to design car prototypes, or just for fun!

Overview
--------

Fastest-lap an open source (`MIT <https://github.com/juanmanzanero/fastest-lap/blob/main/LICENSE>`_) written in C++, and it is usable from any scripting language such as Python and MATLAB.
